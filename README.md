# eslint-config-jochristianto-base

ESLint configuration for Jonathan's JS projects.

## Setup

1. Install
   ```sh
   $ yarn add eslint-config-jochristianto-base --dev
   ```
2. Add to the project's eslint config
   ```sh
   {
       "extends": [
           "jochristianto-base"
       ]
   }
   ```
